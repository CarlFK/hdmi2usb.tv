Getting Help

If you need help with anything, from getting a device running to wanting to develop on HDMI2USB, please get in touch:

  • Sign up to our mailing list: https://groups.google.com/forum/#!forum/hdmi2usb
  • Ask a question on IRC: #timvideos on irc.freenode.org

  • Ping us at @timvideosus on Twitter

  • How to generate a bug report script

Some good pages to look for tips about writing good questions can be found below (TLDR; demonstrate you’ve tried to figure it out first, and provide as much information as possible):

  • How To Ask Questions The Smart Way - Eric Steven Raymond
  • How To Ask Questions The Smart Way by Eric S. Raymond
  • How To: Ask Questions The Smart Way - C++ Articles
